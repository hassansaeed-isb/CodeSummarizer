Analyzing the Limitations of AI-based Code Documentation

Research Project Dashboard

This interactive dashboard showcases the results from our research on the limitations of AI-based code documentation tools like CodeT5 in capturing business logic and domain context.

Our study evaluates CodeT5, a state-of-the-art model for code documentation, finding that while it achieves high accuracy on technical details, it significantly underperforms when capturing business logic and safety implications.
